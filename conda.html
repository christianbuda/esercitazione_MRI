🐍 Fase Finale: VS Code e Python

Per l'esercitazione scriveremo ed eseguiremo del codice Python per estrarre le serie temporali dalle risonanze e costruire matrici di connettività. Per farlo in modo comodo useremo Visual Studio Code (l'editor) e Miniconda (il motore Python).

🚨 AVVISO FONDAMENTALE: FAI QUESTO A CASA!

In aula non avremo accesso a internet per scaricare pacchetti. Devi completare TUTTI i passaggi di questa pagina a casa, inclusa l'installazione finale dei pacchetti con pip. Se non lo fai, il tuo computer non potrà eseguire il codice in classe!

1. Installa Visual Studio Code

🖥️ Utenti Windows

  1. Vai sul sito code.visualstudio.com, scarica l'installer per Windows e installalo normalmente.
  2. Apri VS Code. Sulla barra a sinistra, clicca sull'icona delle Estensioni (quella con i 4 quadratini).
  3. Cerca l'estensione "WSL" (di Microsoft) e clicca su Install. Questo è essenziale per permettere a VS Code di comunicare con il tuo Ubuntu!
  4. Cerca anche le estensioni "Python" e "Jupyter" e installale.

🍎 Utenti macOS

  1. Vai sul sito code.visualstudio.com e scarica la versione per Mac.
  2. Trascina l'app estratta nella cartella Applicazioni.
  3. Apri VS Code. Sulla barra a sinistra, clicca sull'icona delle Estensioni (quella con i 4 quadratini).
  4. Cerca le estensioni "Python" e "Jupyter" e clicca su Install.

2. Installa Miniconda

Ora installiamo Python. Attenzione: chi usa Windows dovrà installare la versione Linux dentro Ubuntu!

🖥️ Utenti Windows (da Terminale Ubuntu)

Apri il tuo terminale Ubuntu. Copia, incolla questo comando e premi Invio (scaricherà l'installer e lo avvierà):

wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh && bash Miniconda3-latest-Linux-x86_64.sh

Premi Invio per scorrere la licenza, scrivi yes per accettarla e premi Invio. Quando ti chiede se vuoi inizializzare Miniconda, scrivi yes e premi Invio.

🍎 Utenti macOS

Vai sulla pagina ufficiale di Miniconda e scarica il Miniconda3 macOS 64-bit pkg (scegli Apple M1/ARM se hai un Mac recente, altrimenti Intel).

Fai doppio clic sul file scaricato e segui le normali istruzioni a schermo (Continua, Accetta, Installa).

Importante: Dopo aver installato Miniconda, devi chiudere e riaprire il terminale (sia su Mac che su Ubuntu). Quando lo riapri, dovresti vedere la scritta (base) prima del tuo nome utente. Significa che Python è attivo!

3. Scarica le Librerie per l'Esercitazione (Serve Internet!)

Ora che tutto è pronto, creiamo un "contenitore" isolato (environment) per la nostra esercitazione e scarichiamo tutti i pacchetti di neuroscienze che ci serviranno. Questo comando è identico per Windows e Mac.

  1. Apri un nuovo terminale (su Windows apri Ubuntu, su Mac apri il Terminale).
  2. Crea un nuovo ambiente chiamato neuro digitando: 
    conda create -n neuro python=3.10 -y
  3. Attiva il nuovo ambiente: 
    conda activate neuro
    (Noterai che la scritta nel terminale è passata da (base) a (neuro))
  4. Installa tutti i pacchetti necessari tramite pip. Questo richiederà un paio di minuti in base alla connessione: 
    pip install jupyter nilearn nibabel matplotlib numpy scipy pandas

4. Come apriremo l'esercitazione in aula

Una volta in classe, non dovrai fare altro che usare questo trucco magico per collegare VS Code al tuo ambiente:

  1. Apri il terminale (Ubuntu per Windows, Terminale per Mac).
  2. Naviga nella cartella dove hai salvato i file dell'esercitazione (es. cd Documents/neuro_lab).
  3. Digita questo comando e premi Invio: 
    code .

Questo aprirà VS Code direttamente nella cartella giusta, pronto per leggere i nostri dati di risonanza magnetica!
