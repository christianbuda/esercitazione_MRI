🚀 Lavorare in aula (macOS)

Infine, ecco i passi da seguire per avere i dati della risonanza e il notebook pronti per l'uso.

  1. Scarica i dati:
    Scarica il file ZIP dell'esercitazione (LINK), salvalo dove preferisci (es. sulla Scrivania) ed estrai la cartella.
    Importante: NON salvare la cartella nei Download perché può dare problemi durante l'esecuzione, spostala!
  2. Apri il Terminale:
    Apri l'app Terminale del Mac. Scrivi cd seguito da uno spazio (non premere ancora Invio!).
  3. Trascina la cartella:
    Trascina la cartella estratta dal Finder direttamente dentro la finestra del Terminale. Il percorso si scriverà da solo. Ora premi Invio.
  4. Avvia VS Code:
    Digita il comando qui sotto e premi Invio per aprire VS Code direttamente in quella cartella: 
    code .
  5. Apri il notebook:
    VS Code si aprirà automaticamente con tutti i tuoi file. Apri il file `.ipynb`, assicurati di selezionare l'ambiente `neuro` in alto a destra e sei pronto a lavorare!
    IMMAGININA VS CODE NELLA CARTELLA CON EVIDENZIATO IL NOTEBOOK DA APRIRE, E IL KERNEL IN ALTO A DESTRA
🎉 TUTTO PRONTO! 🎉
