🐍 Fase Finale: VS Code e Python (macOS)

Per l'esercitazione useremo Visual Studio Code per leggere il codice e Miniconda per farlo funzionare.

1. Installa Visual Studio Code

  1. Vai sul sito code.visualstudio.com e scarica la versione per Mac.
  2. Trascina l'app estratta nella tua cartella Applicazioni.
  3. Apri VS Code. Sulla barra a sinistra, clicca sull'icona delle Estensioni (quella con i 4 quadratini).
  4. Cerca le estensioni "Python" e "Jupyter" e premi "Install".

2. Installa Miniconda in automatico

Apri il Terminale di macOS. Incolla questo comando e premi invio:

cd ~ && curl -O https://repo.anaconda.com/miniconda/Miniconda3-latest-MacOSX-arm64.sh && bash ./Miniconda3-latest-MacOSX-arm64.sh

Premi Invio per scorrere la licenza, una volta in fondo scrivi yes per accettarla e premi Invio.

Premi invio per installare Miniconda nella directory di default. Quando ti chiede se vuoi inizializzare Miniconda, scrivi yes e premi invio.

Importante: Dopo aver installato Miniconda, chiudi e riapri il Terminale. Quando lo riapri, dovresti vedere la scritta (base) all'inizio della riga. Significa che Python è attivo!

3. Scarica le Librerie per l'Esercitazione

Nel terminale Ubuntu appena riaperto esegui questi comandi per accettare i termini di servizio e scaricare i pacchetti python:

conda tos accept --override-channels --channel https://repo.anaconda.com/pkgs/main
conda tos accept --override-channels --channel https://repo.anaconda.com/pkgs/r

Poi, crea l'ambiente per la nostra esercitazione eseguendo questi comandi, uno alla volta:

  1. Crea l'ambiente: 
    conda create -n neuro python=3.10 -y
  2. Attivalo: 
    conda activate neuro
    (la scritta (base) all'inizio della riga dovrebbe essere cambiata in (neuro))
  3. Installa i pacchetti (ci vorrà qualche minuto): 
    pip install jupyter nilearn nibabel matplotlib numpy scipy pandas
