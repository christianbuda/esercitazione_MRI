🖥️ Windows Users: Installing WSL

Because many development tools are built for Linux, we need to install the Windows Subsystem for Linux (WSL). This allows you to run a Linux environment directly inside Windows.

  1. Open PowerShell as Administrator:
    • Click your Start menu and type PowerShell.
    • Right-click "Windows PowerShell" and select Run as administrator.
    • Click "Yes" if prompted by User Account Control.
  2. Run the install command:

    Copy the following text, paste it into your PowerShell window, and press Enter:

    wsl --install

    Note: This will install Ubuntu as your default Linux distribution, which is exactly what we want.

  3. Restart your computer:

    Once the installation finishes, restart your PC to apply the changes.

  4. Create your Linux user account:

    After restarting, a Linux terminal window will pop up automatically. It will ask you to create a UNIX username and password.

    Important: When you type your password, nothing will show up on the screen (no asterisks). This is normal in Linux! Just type it carefully and press Enter.

🎉 You're done! You now have a full Linux environment running inside Windows.

🍎 macOS Users

Good news! macOS is built on a Unix-like operating system natively, which means you already have a terminal environment ready to go.

You do not need to install WSL. You can skip this step entirely and proceed to the next part of the course materials!
