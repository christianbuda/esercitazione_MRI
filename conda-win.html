🐍 Fase Finale: VS Code e Python (Windows)

Per l'esercitazione useremo Visual Studio Code per leggere il codice e Miniconda per farlo funzionare.

1. Installa Visual Studio Code

  1. Vai sul sito code.visualstudio.com, scarica l'installer per Windows e installalo normalmente.
  2. Apri VS Code. Sulla barra a sinistra, clicca sull'icona delle Estensioni (quella con i 4 quadratini).
  3. Cerca l'estensione "WSL" (di Microsoft) e clicca su Install. Questo è fondamentale per permettere a VS Code di comunicare con Ubuntu!
  4. Cerca anche le estensioni "Python" e "Jupyter" e installale.

2. Installa Miniconda (dentro Ubuntu)

Apri il tuo terminale Ubuntu. Copia, incolla questo comando e premi Invio:

cd ~ && wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh && bash ./Miniconda3-latest-Linux-x86_64.sh

Premi Invio per scorrere la licenza, una volta in fondo scrivi yes per accettarla e premi Invio.

Premi invio per installare Miniconda nella directory di default. Quando ti chiede se vuoi inizializzare Miniconda, scrivi yes e premi invio.

Importante: Dopo aver installato Miniconda, chiudi e riapri il terminale Ubuntu. Quando lo riapri, dovresti vedere la scritta (base) all'inizio della riga. Significa che Python è attivo!

3. Scarica le Librerie per l'Esercitazione

Nel terminale Ubuntu appena riaperto, crea l'ambiente per il nostro laboratorio eseguendo questi tre comandi, uno alla volta:

  1. Crea l'ambiente: 
    conda create -n neuro python=3.10 -y
  2. Attivalo: 
    conda activate neuro
  3. Installa i pacchetti (ci vorrà qualche minuto): 
    pip install jupyter nilearn nibabel matplotlib numpy scipy pandas
