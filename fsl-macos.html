🧠 Installazione di FSL

FSL (FMRIB Software Library) è un toolbox molto popolare per l'analisi dei dati di risonanza magnetica.

  1. Apri il tuo Terminale:

    Apri l'app Terminale. Puoi trovarla usando Spotlight (Cmd + Spazio) e digitando "Terminale", oppure navigando nel Finder in Applicazioni > Utility > Terminale.

  2. Esegui il comando di installazione:

    Copia l'intero comando qui sotto, incollalo nel tuo terminale e premi Invio:

    cd ~
curl -Ls https://fsl.fmrib.ox.ac.uk/fsldownloads/fslconda/releases/getfsl.sh | sh -s
  3. Attendi il completamento:

    Ora ci vuole pazienza. Lo script scaricherà e installerà tutto il necessario. Se hai una connessione veloce ci vorranno circa 10-15 minuti, altrimenti potrebbe volerci di più. Il terminale mostrerà l'avanzamento dell'installazione.

  4. Verifica il successo:

    Leggi attentamente l'output finale. Se tutto è andato a buon fine, vedrai questo messaggio:

    FSL successfully installed
    Importante: Una volta vista questa scritta, chiudi completamente e apri una nuova finestra del terminale per applicare le modifiche.
❌ Non vedi il messaggio di successo?
Se si sono verificati errori e l'installazione è fallita:
  1. Salva una copia di tutto il testo che l'installatore ha stampato nel terminale.
  2. Cerca nella tua cartella home un file chiamato fsl_installation_<data>.log (dove <data> è la data odierna).
  3. Invia questi dati al docente o all'assistente per ricevere supporto!

🎉 Ottimo lavoro! FSL è ora installato sul tuo sistema.
